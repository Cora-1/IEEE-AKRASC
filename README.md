A modern, high-performance, single-page website built for the **All Kerala Robotics & Automation Society Conclave (AKRASC)**. 

Moving away from standard corporate templates, this site utilizes a custom "Cyber-Robotics" aesthetic. It is engineered to load instantly, scale flawlessly on mobile devices, and directly target innovators, builders, and disruptors in the robotics space.

## 🚀 Features

* **Dark Cybernetic UI:** Deep space backgrounds, neon typography (Space Grotesk), and grid-based abstract layouts.
* **Fully Responsive:** Built with mobile-first principles ensuring perfect rendering across all devices.
* **Zero-Build Architecture:** Utilizes Tailwind CSS via CDN for instant deployment without complex Node.js build steps.
* **Integrated Sections:**
    * `Initialize (About):` High-level overview of the conclave's technical capabilities.
    * `Lead (Coordinator):` Dedicated profile highlighting technical authority.
    * `Access Codes (Tickets):` Pricing tiers for Project Expo and Full Override passes.

## 🛠️ Tech Stack

* **Markup:** HTML5
* **Styling:** Tailwind CSS (CDN)
* **Typography:** Google Fonts (Space Grotesk)
* **Icons:** Heroicons (SVG)
